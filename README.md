## Installation (Fork / Clone)

Download and install all dependencies
```bash
npm install
yarn install
```

Rebuild sqlite
```bash
npm run rebuild
```

Start dev with hot reload
```bash
npm run dev
```

Build for production (Electron)
```bash
npm run build
```
